Penjelasan Gerbang Logika

Gerbang AND

Gerbang AND menghasilkan nilai benar (1) jika kedua input bernilai benar. Jika salah satu atau kedua input bernilai salah (0), outputnya salah.

Gerbang OR

Gerbang OR menghasilkan nilai benar (1) jika salah satu atau kedua input bernilai benar. Output hanya salah (0) jika kedua input bernilai salah.

Gerbang NOT

Gerbang NOT adalah gerbang inversi. Outputnya akan menjadi kebalikan dari input. Jika inputnya benar (1), outputnya salah (0), dan sebaliknya.

Gerbang XOR

Gerbang XOR menghasilkan nilai benar (1) jika satu dan hanya satu dari kedua input bernilai benar. Jika kedua input sama (baik 0 atau 1), outputnya salah.

Gerbang NAND

Gerbang NAND adalah kebalikan dari gerbang AND. Outputnya akan salah (0) hanya jika kedua input bernilai benar (1). Jika tidak, outputnya benar (1).

Gerbang XNOR

Gerbang XNOR adalah kebalikan dari gerbang XOR. Outputnya akan benar (1) jika kedua input bernilai sama, dan salah (0) jika kedua input berbeda.
